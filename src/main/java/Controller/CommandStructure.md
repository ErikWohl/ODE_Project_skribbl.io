# Command structure
command|values<br>

command ... 3 Zeichen langer Befehl<br>
| ... separator, wird nicht mitgeschickt<br>
values ... <br> werden je nach Befehl anders abgearbeitet
## Serverside
•	STARTGAME<br>
•	STARTROUND<br>


## Clientside
•	Message: MSG|text<br>
•	Drawing: DRW|x1;y1;x2;y2<br>
•	Start game: STG|<br>
